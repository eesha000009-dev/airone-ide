# Airone IDE

Robotics Programming Environment based on Arduino IDE 2.x with .airo language support.

## Features
- Full .airo language support (syntax highlighting, snippets)
- Built-in Airo Compiler (.airo → C++/Arduino transpilation)
- SENSE → THINK → ACT programming model
- WebSocket communication with AI Brain
- Sync to Airone AI Backbone integration
- ESP32 target support

## Download
Check the [Releases](https://github.com/eesha000009-dev/airone-ide/releases) page for Windows builds.

## Requirements
- Windows 10/11 (x64)
- Python 3.8+ (for .airo compilation)
- ESP32 board support (via Arduino Board Manager)
